## Tech Stack

- Next.js
- Typescript
- Markdown Content
- Firebase Hosting
- Google Analytics

## How to Run Locally

1. Install packages.

```
npm install
```

2. Start the application (optional: [define a content mock](./content/mock/README.md)).

```
npm run dev
```
